 Streetwear Store

A modern, responsive e-commerce website inspired by Formula 1 culture and streetwear fashion. Built with a bold UI, smooth navigation, and category-based shopping to deliver a race-ready digital experience.

## ✨ Features

* 🧢 **Category-based shopping**

  * Tees
  * Jackets
  * Caps
  * Accessories
  * Limited Editions
* 🛒 **Cart system**
* 🔐 **Login & Sign Up pages**
* 📱 **Responsive design** (mobile, tablet, desktop)
* 🎨 **F1-inspired UI** with bold colors and modern layout
* ⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

* **Frontend:**

  * HTML5
  * CSS3
  * JavaScript
  * Reactjs
